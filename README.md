## Descripción del sistema

Este proyecto consiste en un sistema básico de gestión de restaurante desarrollado en Python utilizando Programación Orientada a Objetos (POO). El sistema permite registrar productos y clientes, almacenarlos dentro del restaurante y mostrar la información organizada en consola.

## Estructura del proyecto

```text
PROYECTO_RESTAURANTE
│
├── README.md
│
└── restaurante_app
    │
    ├── main.py
    │
    ├── modelos
    │   ├── producto.py
    │   └── cliente.py
    │
    └── servicios
        └── restaurante.py
```

## Explicación de los archivos

* producto.py: contiene la clase Producto.
* cliente.py: contiene la clase Cliente.
* restaurante.py: contiene la clase Restaurante que administra los productos y clientes.
* main.py: es el archivo principal donde se crean los objetos y se ejecuta el programa.

## Reflexión

La modularización permite organizar el software en diferentes archivos según sus responsabilidades. Esto facilita la comprensión, el mantenimiento y la reutilización del código. Además, la Programación Orientada a Objetos ayuda a representar entidades del mundo real mediante clases y objetos, haciendo que el desarrollo sea más ordenado y eficiente.
